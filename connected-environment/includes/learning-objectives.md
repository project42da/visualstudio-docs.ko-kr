이 가이드에서는 다음 방법을 알아봅니다.

1. 개발에 최적화된 Azure에서 Kubernetes 기반 환경을 만듭니다.
1. VS Code 및 명령줄을 사용하여 컨테이너에서 코드를 반복해서 개발합니다.
1. 팀 환경에서 코드를 생산적으로 개발하고 테스트합니다.
