# <a name="c28209"></a>C28209
C28209 경고: 기호에 대 한 선언에 충돌 하는 선언

 이 경고는 잘못 생성된 주석 선언을 나타냅니다. 일반적인 사용의 경우 이 경고가 발생하지 않습니다.
