# <a name="c28243"></a>C28243
C28243 경고: 주석이 함수 매개 변수를 더 필요한 것 보다 역참조 실제 형식 주석이 허용  
  
 주석에서 `__deref` 연산자의 수는 매개 변수 형식으로 정의된 포인터 수준 수 이상입니다. 주석 또는 참조된 매개 변수의 포인터 수준 수를 변경하여 이 경고를 해결하십시오.
