# <a name="c28280"></a>C28280
C28280 경고: 기호는 'end'이 없는 일치 하는 '시작'

 주석 언어가 begin 및 end 지원 (`{` 및 `}` c에서) 구문 및 쌍의 균형이 합니다. 매크로를 사용하는 경우 이 상황을 방지할 수 있습니다.
