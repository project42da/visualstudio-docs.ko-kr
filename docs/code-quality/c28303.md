# C28303
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

경고 C28303: C\+\+ 참조 매개 변수 \<parameter\_name\>의 경우 모호한 `_Deref_` 연산자가 \<annotation\>에서 발견되었습니다.  
  
 이 경고는 경고 C28302 와 유사하고 `_Deref_` 의 추가 수준이 매개 변수에 사용 될 때 보고됩니다.  
  
 SAL2은 참조 매개 변수를 처리할 때 `_Deref_` 의 추가적인 수준의 사용을 요구하지 않습니다.  이 특정 주석은 어느 역참조 수준을 주석 처리 할지 모호 합니다.  이것은 주석처리 될 특정 개체를 참조하기 위해 `_At_` 을 사용해야 할 수도 있습니다.  
  
## 예제  
 다음 코드에서는 `__deref_out_ecount(n)` 의 사용이 모호하기 때문에 이 경고를 생성됩니다.  
  
```  
void ref(__deref_out_ecount(n) int **&buff, int &n)  
```  
  
 위의 주석은 다음 중 하나로 해설될 수 있습니다:  
  
-   정수의 배열\(n\) 포인터에 대한 참조 \(SAL1 해석\)  
  
-   배열\(n\) 정수의 포인터에 대한 참조 \(SAL2 해석\)  
  
 다음 중 하나는 이 경고를 해결할 수 있습니다.  
  
```  
void ref(_Out_writes_(n) int **&buff, int &n)  
// or  
_At_(*buff), _Out_writes(n)) void ref(int **&buff, int &n)  
  
```
