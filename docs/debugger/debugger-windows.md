# 디버거 창
[!INCLUDE[vs2017banner](../code-quality/includes/vs2017banner.md)]

프로그램을 디버그하는 동안 대부분의 디버거 창을 열 수 있습니다. 디버거 창의 목록을 보려면 중단점을 설정하고 디버깅을 시작합니다. 중단점을 적중하고 실행이 중지되면 **디버그 \/ Windows**를 클릭합니다.
