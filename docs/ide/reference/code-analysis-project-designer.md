# 프로젝트 디자이너, 코드 분석
[!INCLUDE[vs2017banner](../../code-quality/includes/vs2017banner.md)]

코드에서 실행할 수 있는 코드 분석 도구가 포함되어 있습니다.  이 도구에서는 Microsoft .NET Framework 디자인 지침에 제시된 프로그래밍 및 디자인 규칙 위반 같은 어셈블리에 대한 정보를 보고합니다.
