# <a name="report-based-on-visible-time-range"></a>표시되는 시간 범위를 기반으로 하는 보고서
프로필 뷰는 현재 표시된 시간 범위 및 채널을 기반으로 하는 보고서를 표시합니다. 다른 데이터 하위 집합에 대한 세부 정보를 보려면 범례에서 항목을 클릭합니다.  
  
 여기의 테이블에서 데이터에 대한 자세한 정보를 찾을 수 있습니다.
