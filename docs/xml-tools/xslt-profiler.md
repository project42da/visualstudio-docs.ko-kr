# <a name="xslt-profiler"></a>XSLT 프로파일러

XSLT 프로파일러는 XSLT 문서를 개발하고 디버깅하는 데 도움이 되는 성능 분석 프로파일러 도구입니다. 측정 하 여 자세한 XSLT 성능 보고서를 만들어 XSLT 코드의 성능 관련 문제를 평가할 수 있습니다. XSLT 프로파일러에는 최대 성능을 요구하는 XSLT 기반 응용 프로그램에 필수적인 XSL 및 XSLT 스타일시트 최적화에 대한 유용한 힌트가 포함되어 있습니다.

XSLT 프로파일러는 Visual Studio의 일부 이며에서 사용할 수는 **XML** 메뉴.

![XSLT 프로파일러](../xml-tools/media/profile-xslt-menu.png)

> [!NOTE]
> XSLT 프로파일러는 Visual Studio 2017의 Enterprise edition에서 사용할 수만 있습니다.
