# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE 오류 코드
다른 스레드에서 될 수 있는 호출자에 게 스크립트 오류를 전파 하는 중입니다. 호스트는 오류 코드는 호출자에 게 전달 해야 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
