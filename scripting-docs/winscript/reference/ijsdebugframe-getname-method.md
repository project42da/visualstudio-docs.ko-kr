# <a name="ijsdebugframegetname-method"></a>IJsDebugFrame::GetName 메서드
스택 프레임의 이름을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```  
HRESULT GetName(  
   BSTR *pName  
);  
```  
  
#### <a name="parameters"></a>매개 변수  
 `pName`  
 [out] 스택 프레임의 이름입니다.  
  
## <a name="return-value"></a>반환 값
