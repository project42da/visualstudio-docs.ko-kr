# <a name="tostring-method-map-javascript"></a>toString 메서드(Map)(JavaScript)
Map의 문자열 표현을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
mapObj.toString()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `mapObj`  
 필수 요소. `Map` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 Map의 문자열 표현입니다.
