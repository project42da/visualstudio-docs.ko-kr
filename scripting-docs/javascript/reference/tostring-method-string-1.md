# <a name="tostring-method-string-1"></a>toString 메서드 (String) 1
문자열의 문자열 표현을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
string.toString()  
```  
  
## <a name="parameters"></a>매개 변수  
 `string`  
 필수 요소. 나타내는 문자열 배열입니다.  
  
## <a name="return-value"></a>반환 값  
 문자열의 문자열 표현입니다.  
  
## <a name="example"></a>예제  
 다음 예제에서는 **toString** 메서드는 문자열을 사용 합니다.  
  
```JavaScript  
var string = "this is a test";  
var strStr = string.toString();  
document.write(strStr);  
  
// Output: this is a test  
  
```
