# <a name="mathpow-function-javascript"></a>Math.pow 함수(JavaScript)
지정된 된 거듭제곱을 수행 하는 기본 식의 값을 반환 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.pow(  
base, exponent)   
```  
  
## <a name="parameters"></a>매개 변수  
 *base*  
 필수 요소. 식의 기본 값입니다.  
  
 *지 수*  
 필수 요소. 식의 지 수 값입니다.  
  
## <a name="example"></a>예제  
 다음 예제에서는 숫자 식 기준 1000을 반환합니다.  
  
```JavaScript  
Math.pow(10,3);  
```
