# <a name="mathtan-function-javascript"></a>Math.tan 함수(JavaScript)
숫자의 탄젠트를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.tan(  
radians  
)  
  
```  
  
## <a name="remarks"></a>설명  
 필요한 `radians` 인수는 라디안 단위의 각도 포함 하는 숫자 식입니다.  
  
 반환 값이 숫자는 인수의 탄젠트 `radians`합니다.
