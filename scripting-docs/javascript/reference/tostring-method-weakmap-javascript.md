# <a name="tostring-method-weakmap-javascript"></a>toString 메서드(WeakMap)(JavaScript)
문자열 표현을 반환는 `WeakMap` 개체입니다.  
  
## <a name="syntax"></a>구문  
  
```JavaScript  
weakmapObj.toString()  
```  
  
#### <a name="parameters"></a>매개 변수  
 `weakmapObj`  
 필수 요소. `WeakMap` 개체입니다.  
  
## <a name="property-valuereturn-value"></a>속성 값/반환 값  
 `WeakMap`의 문자열 표현입니다.
