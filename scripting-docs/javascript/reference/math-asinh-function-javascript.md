# <a name="mathasinh-function-javascript"></a>Math.asinh 함수(JavaScript)
숫자의 역쌍곡 사인을 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.asinh(x)   
```  
  
## <a name="remarks"></a>설명  
 `x`가 `NaN`인 경우, 결과는 `NaN`입니다.  
  
 **적용 대상**: [Math 개체](../../javascript/reference/math-object-javascript.md)
