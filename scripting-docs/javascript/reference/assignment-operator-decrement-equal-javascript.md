# <a name="assignment-operator--javascript"></a>할당 연산자(=)(JavaScript)
변수에 값을 할당합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>매개 변수  
 `result`  
 모든 변수입니다.  
  
 `expression`  
 임의의 숫자 식입니다.  
  
## <a name="remarks"></a>설명  
 = 연산자를 포함 하는 식 값을 갖도록 다른 연산자 처럼 동작 합니다. 즉, 대입 연산자를 다음과 같이 연결할 수 있습니다: `j = k = l = 0`합니다. 이 경우 `j`, `k`, 및 `l` 값이 0 인 합니다.
