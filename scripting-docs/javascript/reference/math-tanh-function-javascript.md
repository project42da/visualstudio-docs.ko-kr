# <a name="mathtanh-function-javascript"></a>Math.tanh 함수(JavaScript)
숫자의 쌍곡 탄젠트를 반환합니다.  
  
## <a name="syntax"></a>구문  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>설명  
 반환 값은 `(exp(x) - exp(-x))/(exp(x) + exp(-x))`와 동일합니다. `x`가 `NaN`인 경우, 결과는 `NaN`입니다.
