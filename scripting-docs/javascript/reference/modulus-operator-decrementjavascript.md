# <a name="modulus-operator--javascript"></a>모듈러스 연산자 (JavaScript)
숫자 식의 값을 다른 숫자 식의 값으로 나누고 나머지를 생성 합니다.  
  
## <a name="syntax"></a>구문  
  
```  
result = expression1 % expression2  
```  
  
## <a name="arguments"></a>인수  
 `result`  
 모든 변수입니다.  
  
 `expression1`  
 임의의 숫자 식입니다.  
  
 `expression2`  
 임의의 숫자 식입니다.
