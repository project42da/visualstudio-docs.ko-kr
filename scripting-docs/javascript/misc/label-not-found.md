# <a name="label-not-found"></a>레이블을 찾을 수 없습니다.
존재 하지 않는 레이블에 대 한 참조를 했습니다. 레이블은 코드의 하지만 지정 된 범위 내에서 블록을 표시 하는 데 사용할 수, 고유 해야 합니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   레이블 이름 철자가 틀리지 있는지 확인 합니다.  
  
-   모든 레이블 참조 (다음 정의 포함) 현재 범위에 정의 된 레이블로 내용이 확인 합니다.
