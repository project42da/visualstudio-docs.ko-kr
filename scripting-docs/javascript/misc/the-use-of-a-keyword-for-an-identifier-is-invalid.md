# 식별자에 대해 키워드를 잘못 사용했습니다.
1048: 식별자에 대한 키워드의 잘못된 사용  
  
 변수, 함수 또는 기타 사용자 지정 식별자의 이름으로 키보드를 사용할 수 없습니다.  
  
 키워드에 대한 자세한 내용은 [JavaScript 예약어](../../javascript/reference/javascript-reserved-words.md) 및 [JavaScript 이후 버전 예약어](../../javascript/reference/javascript-future-reserved-words.md)를 참조하세요.
