# <a name="expected-3939"></a>예상 &#39;: &#39;
삼항 조건부 연산자를 사용 하 여 식을 만들 하려고 하지만 두 번째 및 세 번째 피연산자 사이 콜론을 포함 하지 않은 있습니다. 삼항 (피연산자 3 개) 조건부 연산자 (true) 두 번째 및 세 번째 (false) 사이 콜론 필요 피연산자.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   두 번째 및 세 번째 피연산자 사이 콜론을 삽입 합니다.
