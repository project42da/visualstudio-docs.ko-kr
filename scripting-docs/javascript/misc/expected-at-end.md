# <a name="expected-39end39"></a>예상 &#39; @end&#39;
조건부로 컴파일된 코드 블록을를 만들려고 했으나 포함 되지 않은  **@end**  문 끝에 있습니다. [!INCLUDE[javascript](../../javascript/includes/javascript-md.md)]문 내에서 포함 하 여 조건에 따라 컴파일할 수 있습니다는  **@if/ @end**  블록입니다.  
  
### <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   해당 추가  **@end**  문.
