# <a name="searching-and-filtering-subscribers-in-visual-studio-subscriptions"></a>Visual Studio 구독에서 구독자 검색 및 필터링
Visual Studio 구독 관리 포털을 사용하면 다양한 조건에 따라 구독자의 특정 하위 집합을 찾을 수 있습니다. 이름, 이메일 주소, 구독 수준 및 기타 다양한 정보를 기준으로 구독자 목록을 필터링할 수 있습니다. 

1.  특정 구독자 그룹을 검색하려면 테이블 위쪽에서 **필터**를 클릭합니다.

    ![구독자 필터링](_img\edit-license\filter-list.png)

2.  검색 조건으로 사용할 수 있는 많은 필드가 표시됩니다. 검색 기준으로 사용하려는 원하는 조건을 선택한 다음, **필터**를 클릭합니다. 선택한 조건과 일치하는 구독자의 목록이 표시됩니다.
![구독자 찾기](_img\search-filter\search-filter-find.png)
